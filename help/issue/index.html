如何手动安装BAS浏览器插件?

步骤一、下载安装文件

点击按钮下载压缩文件

步骤二、打开“扩展程序”页面

点击 Chrome 浏览器右上角菜单栏，从“更多工具”中打开“扩展程序”

步骤三、打开开发者模式

找到扩展程序页面右上角的开发者模式开关，点击打开

步骤四、安装扩展程序

将下载的BAS插件zip包拖入Chrome扩展程序页面安装

为什么需要Metamask插件?

为什么需要Metamask插件?

MetaMask是一款在浏览器上使用的区块链帐户管理插件，只需要在谷歌或火狐浏览器添加对应的扩展程序即可，非常轻量级，使用起来也非常方便,并且安全.

您所购买的域名都存在您自己的帐户下,未经您的授权,任何人或机构都无法侵吞或通过技术手段占有您私有的域名资产.因此你需要通过Metamask管理您的帐户.

如何安装MetaMask并创建账号?

安装 MetaMask

如果您使用的是Chrome浏览器或FireFox浏览器, 你可以搜索 ＂MetaMask＂ 或点击链接 去官网 https://metamask.io/ 下载扩展程序,也可以点击下面按钮到浏览器扩展商店下载.

Chrome Web Store Download Firefox ADD-ONS Download

下面介绍如何安装MetaMask

请在Chrome网页版商店点击 'Add to Chrome' 按钮

网页将弹出一个窗口，询问是否要添加MetaMask，确认内容后请点击'ADD EXTENSION'。

完成安装后请点击‘Get Started’

如果您是第一次使用MetaMask，为了创建账户，请点击右边选项创建钱包

选择是否同意协助MetaMask改善产品

请设置密码

设置秘密备份短语，记录并按顺序点击以确认短语

创建成功

如何免费在测试网申请域名，并且使用BAS系统？

使用BAS域名系统需要以下几项准备工作：
  • 你需要使用谷歌浏览器（Chrome）或者火狐浏览器（Firefox）来操作本网站
  • 你需要安装MetaMask浏览器插件
  • 你需要安装BAS插件或者下载BAS浏览器来体验BAS域名解析服务
注册域名

在域名申请菜单下，查询你想要注册的域名，确认是否已经被注册。如果没有被注册，你可以去注册它

选择购买期限，点击注册

此时会弹出MetaMask授权交易窗口,点击确认后,出现正在授权信息 ,注意此时不要关闭或刷新页面,否则注册取消

授权成功后,再次弹出MetaMask确认交易界面,此时点击确认,提交域名到BAS链上.

当注册信息显示已成功时,代表此域名已经成为你的个人域名资产,在 [ 我的钱包 ] 里可以查看与管理.

在[ 我的钱包 ] 中，你可以配置域名对应的IP地址和其他信息

如果你设置了IP，你可以使用配置好的域名，通过插件或浏览器在地址栏输入 “httpb://你的域名” 来访问
